## Getting Started

1. Setting your Redis connection info in `Constants.java`

```java
// redis
public static final String HOST = "localhost";
public static final int PORT = 6379;
public static final String PASSWORD = "123456";
```

2. Running `com.qiujie.util.DataLoader` to load data into Redis (⚠️ Run before first start example)
3. There are some examples in the `com.qiujie.example` package
